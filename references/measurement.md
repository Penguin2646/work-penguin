# measurement — 성공기준 → 측정법 매핑

1단계에서 고른 기준에 맞춰 5단계에서 효과를 잰다. 입문자에겐 **쉬운 숫자**로.

## ⏱️ 시간절약

- Before: 1회 소요시간 × 빈도 (예: 3시간 × 월 4회 = 월 12시간)
- After: 새 방식 1회 소요시간 × 빈도
- 효과: **월 절감시간** = (Before-After) × 빈도
- 표기: "회당 3시간 → 40분, 월 약 9시간 절감"
- 과장 금지. 모르면 범위로("대략 1~2시간").

## 📊 정량지표

- 무엇을 셀지 사용자와 합의: 처리 건수, 오류율, 단계 수, 클릭 수, 리드타임 등
- Before/After 둘 다 추정치라도 기록 (출처·가정 명시)
- 표기: "수기 단계 12 → 3, 오류 가능지점 5 → 1"

## ✨ 퀄리티

- 시간이 아니라 **산출물 자체**를 비교
- 체크 항목(업무별로 택): 정확도 · 구조/가독성 · 일관성 · 디자인 완성도 · 재사용성
- Before/After 산출물을 나란히 보여주고 "이 점이 좋아졌어요" 3가지
- 가능하면 HTML 웹 뷰로 Before/After 시각화

## 후속 확인 (루프의 핵심)

측정은 1차 **추정**이다. 진짜 검증은 다음에:
- 다음 세션 시작 시 스코어보드의 "후속확인" 빈 항목을 먼저 확인
- "지난번 OO, 실제로 시간 줄었어요?" → ✅(효과 확인) / ⚠️(기대만큼 아님, 원인+보정)
- ⚠️면 후보를 다시 잡아 루프를 돈다 = **기존 대비 유의미한 결과를 만들어 가고 있는지 체크**
